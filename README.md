# ℹ  Sobre o Projeto
<p align="justify"> 
Como parte do ProviHack Pra_Todos, buscamos desenvolver uma solução para questões relacionadas à diversidade. Escolhemos como tema a comunidade LGBT+. Para entender as necessidades e desafios desse grupo, elaboramos dois tipos de pesquisa: um formulário respondidas por 32 pessoas e entrevistas semi-estruturadas conduzidas com 7 participantes.

Através das pesquisas, identificamos dificuldades relacionadas a busca por empregos. Sendo assim, elaboramos uma plataforma onde pessoas da comunidade LGBT+ tivessem contato direto com empresas com politicas de inclusão e ambiente diverso e harmônico.em produtos de uso pessoal, doméstico ou até mesmo para decoração.
</p>


**- Experiência dos usuários (Back-End)**

O usuário poderá cadastrar | Alterar | Atualizar | Deletar: Usuários, Empresas e Vagas.
